# 💰 Expense Tracker

## 📌 Overview
A web application to track daily expenses and manage personal budgets.

## 🛠 Tech Stack
- Python / Django
- JavaScript
- SQLite / PostgreSQL

## ✨ Features
- Add and manage expenses
- Categorize spending
- View expense history
- Simple and clean UI

## 📂 Installation

git clone https://github.com/mayurk11/EXPENCE_TRACKER
cd EXPENCE_TRACKER
pip install -r requirements.txt
python manage.py runserver
